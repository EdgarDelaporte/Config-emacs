                                                                                                                tuhdo Mod CodyChan • 8 years ago

                                                                                                                There's none. I run gdb-many-windows in another frame and switch back and forth between code frame and debugger frame with "C-x 5 o". Or you may want to switch using by names if you have many frames using command "select-frame-by-name", default to "C-x 5 s". After finishing your debugging session, simply close the debugger frame with "C-x 5 0".
